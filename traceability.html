Traceability System

A four-stage, documented chain-of-custody system that ensures every certified life diamond can be traced back to its biological source.

The Four Stages

Each stage generates documentation that is reviewed during ILDA's annual audit.

Stage 01

Carbon Source Receipt

The biological sample (hair, fur, or ashes) is received by the laboratory with a unique chain-of-custody ID. The sample is weighed, photographed, and logged in a tamper-evident record system.

  • Sample weight recorded
  • Photographic documentation
  • Unique ID assigned to the order

Sample Registration

Laboratory Processing

Stage 02

Laboratory Processing

The carbon is extracted and purified, then subjected to High-Pressure High-Temperature (HPHT) synthesis. Process parameters — temperature, pressure, duration — are logged and associated with the unique order ID.

  • Carbon extraction process logged
  • HPHT parameters recorded
  • Technician signature/ID
Stage 03

Gemological Grading

The finished diamond is submitted to an independent gemological laboratory (IGI or GIA) for grading. The grading report includes the unique ILDA traceability reference, linking the gemological data to the carbon source record.

  • Independent IGI/GIA grading
  • 4C parameters documented
  • ILDA traceability reference included

Gemological Grading

ILDA Certification

Stage 04

ILDA Certification

ILDA reviews the complete documentation chain and issues a traceability certificate with a unique verification code. The certificate is entered into our public verification database and can be queried by anyone at any time.

  • Complete documentation review
  • Unique certificate number issued
  • Public verification database entry

What Documentation Is Verified?

ILDA's annual audit reviews the following documents for each certified member laboratory.

Chain of Custody

Sample receipt logs, ID assignment records, and transfer documentation between process stages.

Process Records

HPHT parameter logs, technician assignments, equipment calibration records, and quality control notes.

Gemological Reports

IGI/GIA grading reports with ILDA traceability references, cross-checked against process records.
